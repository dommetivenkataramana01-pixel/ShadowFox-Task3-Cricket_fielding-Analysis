🏏 Cricket Fielding Performance Analysis (RCB vs CSK | IPL 2026)
📌 Project Overview

This project was completed as Task 3 during the ShadowFox Data Science Internship.

The objective was to collect and organize ball-by-ball fielding data from an IPL match and evaluate individual fielding performances using a predefined performance scoring formula.

🎯 Objective
Collect detailed fielding events from a complete IPL innings.
Record every meaningful fielding action.
Categorize fielding efforts.
Calculate individual player performance scores using the provided scoring matrix.
📊 Dataset Features

The dataset contains:
